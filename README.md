Student Data Analyzer

📌 Project Description

This is a beginner-friendly Python project that analyzes student marks using basic programming concepts.

🚀 Features

- Calculate average marks
- Find highest marks
- Filter marks greater than a threshold (e.g., >60)

🛠️ Technologies Used

- Python

💡 What I Learned

- Working with lists in Python
- Using loops and conditions
- Basic data processing logic

📈 Future Improvements

- Take input from users
- Read data from CSV files
- Apply Machine Learning algorithms

▶️ How to Run

1. Install Python
2. Run the file:
   python student_analyzer.py
